콘솔은 어떻게 여나요? 열기/닫기
콘솔은 (~ , `) 키로 열 수 있습니다.

이때, 키를 게임내에서 입력했는데도 불구하고 콘솔이 안켜지면 아래와 같이 하시면됩니다.

1. 설정

2. 게임 클릭

3. 게임 - 개발자 콘솔 사용(~) - 사용

4. 게임내에서 ~를 입력하여 아래 사진과 같은 콘솔창이 켜지면 성공

명령어 목록

모든 명령어는 !와 / 둘 다 가능합니다.

일반 사용자 명령어

명령어 설명 예시
!admins 접속중인 어드민 목록 !admins
!emote 춤, 감정표현 사용 !emote, !dance
!getvip VIP 구매 링크 안내 !getvip
!kev 방탄복을 구매합니다. !kev, !kevlar
!mapcooldowns 쿨다운인 맵 목록을 출력합니다 !mapcooldowns
!noshake 화면 흔들림 효과를 제거합니다. !noshake
!rtv 맵 변경에 동의합니다. !rtv
!tp 3인칭 <-> 1인칭 전환 !tp, !thirdperson

후원자 전용 명령어

후원자 권한이 있는 유저만 사용할 수 있습니다.

명령어 설명 예시
!modelcolor 내 캐릭터의 색깔을 정합니다. !modelcolor 255 255 255
!tag 채팅 태그를 설정합니다 !tag 빛이나는솔로
!tagcolor 채팅 태그의 색깔을 정합니다. !tagcolor [0~15]
!vip VIP 남은 기간을 확인합니다. !vip
모든 명령어를 확인하려면 디스코드를 확인해주세요.
